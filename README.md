# EAGitLab
A PowerShell module to manage Users, Groups, and Projects in GitLab. Uses the GitLab API to perform all actions.

Provides the following functions:

Get-AllGitGroups

Get-AllGitGroupsAndMembers

Get-AllGitUsers

Get-GitGroupID

Get-GitUserID

New-GitGroup

New-GitGroupAccessRequest

New-GitUser

Add-GitGroupMember
